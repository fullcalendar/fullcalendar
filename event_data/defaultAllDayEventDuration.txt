
defaultAllDayEventDuration
==========================

A fallback duration for all-day [Event Objects](../event_data/Event_Object) without a specified `end` value.

<div class='spec' markdown='1'>
[Duration](../utilities/Duration), *default*: `{ days: 1 }`
</div>

If an event does not have an `end` specified, it will appear to be this duration when rendered.

The actual `end` of the event will remain unset unless [forceEventDuration]() has been set to `true`.

This setting only affects events with `allDay` equal to `true`.
For timed events, use [defaultTimedEventDuration]().
